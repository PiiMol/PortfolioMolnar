INFORMÁTICA APLICADA

En el trabajo práctico se desarrolló
una base de datos para una página de videojuegos,
utilizando MySQL para gestionar y organizar los datos.
La página web se implementó con HTML, CSS y JavaScript,
proporcionando una interfaz intuitiva y atractiva. Los
datos de los videojuegos, incluyendo títulos, géneros,
plataformas y fechas de lanzamiento, se almacenan y ordenan
eficientemente en la base de datos. Esto permite a los
usuarios buscar, filtrar y acceder fácilmente a la
información relevante sobre los videojuegos disponibles.
